<b>A javascript project for storing notes virtually in web browser with local storage and minimalistic design.</b>
<br>
<br>
<hr>
<p>Features:</p>
<ul>
  <li>Minimalistic design</li>
  <li>Local storage</li>
  <li>Responsive</li>
</ul>
<hr>
<p>How to use:</p>
<ul>
  <li>Clone the repository</li>
  <li>Open index.html in your browser</li>
</ul>
